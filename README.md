# Gold Price Prediction

This repository contains the saved model artifacts and notebook files for a gold price forecasting project.

## Files

- `final_gold_price_forecasting_model.pkl` - trained forecasting model
- `gold_price_scaler.pkl` - fitted feature scaler
- `feature_columns.pkl` - feature order expected by the model
- `app.py` - starter Streamlit app for manual or CSV prediction
- `requirements.txt` - Python dependencies for the app
- `Capstone_2.ipynb` and `__notebook_source__.ipynb` - project notebooks/source

## Run The App

```bash
pip install -r requirements.txt
streamlit run app.py
```
